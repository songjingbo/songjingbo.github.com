# 框架结构

Spring 框架由7个模块组成

```
    ------------------------------
    | AOP | ORM | WEB      | MVC |
    |     |----------------|     |
    |     | DAO | Context  |     |
    ------------------------------
    |     Spring Core            |
    ------------------------------
```

<!--more-->

每个模块的功能如下

Core: 主要组件是 BeanFactory，实现控制反转(IOC), 通过配置解耦程序代码

Context: 上下文配置, 包括企业服务，例如 JNDI、EJB、电子邮件、国际化、校验和调度功能

AOP: 提供AOP支持

DAO：DB访问

ORM：提供ORM工具，其中包括 JDO、Hibernate

Web: 提供Web应用程序上下文. 支持与 Jakarta Struts 的集成

MVC: MVC框架
